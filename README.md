The topic raised is not relevant to WCAG3 and will be closed. Contributor WCAG3 does not rely on, or even encourage validators to be used for conforming to the standard, so any references to legal requirements are not relevant. 
An issue could be raised on the validator if it hasn't been updated to recognize modern CSS, but this is barely related to accessibility, and is not required by WCAG.
